Background

We worked hard to make it so that all our respective websites were as awesome as could be; but first what is a jigsaw?

So, what are they?

'Jigsaws' were projects completed in early December that required us to learn a CSS topic and then make a challenge that hopefully teaches other students about said topic.

Perseverance

Maintaining a strong work flow took a lot of discipline. We were given a lot of free reign with how we wanted to make our jigsaws, which came with creative freedom and many challenges to overcome.

Reflection

We all have come since August 31; within the span of 3 months many of us went from having virtually zero knowledge of programming to being able to make robust coding puzzles. This is why we want to share our progress with you.

Ransom Notes

A Jigsaw to teach CSS Units and Transforms

Difficulty: Easy

In this Jigsaw you will be required to make a "Ransom Note" using primiarily CSS Units and CSS Transforms. This means that you'll be inserting values to match shapes of letters to your model based on the instructions. Remember, have fun!

  • Author: Mason Q.
  • Topic: CSS Units and CSS Transforms

Difficulty: Hard

Reflections

A CSS Jigsaw to teach CSS Image Reflection

Difficulty: Easy

In this Jigsaw you will be reflecting the images given to you using CSS Image Reflections. Remember, have fun!

  • Authors: Dylan G. and Ethan J.

Beyond The Wall

A Jigsaw to teach CSS Backgrounds.

Difficulty: Intermediate

Using CSS Backgrounds you will make a crosshair over your mouse that shows a fire-breathing snowman on click.

  • Authors: Owen S. and Daniel M.
  • Category: CSS Backgrounds

Hidden Message

A jigsaw to teach CSS Z-index.

Difficulty: Intermediate

In this jigsaw you'll be using z-index to reveal a hidden messsage. I'll take the liberty of making this a little less arbitrary and say that the message reveals the mystery of the murder in the Westfield Manor... anyways, have fun! Oh, and I'm sure you'll receive a handsome reward for solving the mystery too! (you won't)

  • Authors: Aeden M. and Chris P.
  • Category: CSS Z-index
